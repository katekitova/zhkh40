# ЖКХ40.РФ на Flask

Полноценный демонстрационный сайт на Flask, собранный по присланным референсам, документам и пользовательским сценариям.

## Что внутри

- главная страница с чат-сценариями по вопросам ЖКХ;
- база знаний с тарифами, материалами по УК/ТСЖ, документами и блок-схемами;
- калькулятор электроэнергии и универсальный расчёт по ресурсам;
- страница образца заявления на перерасчёт;
- форма заполнения заявления с выгрузкой в `DOCX` и печатной версией для `PDF`;
- отдельные шаблоны для хедера, футера, CSS и JavaScript;
- локальная библиотека `docx/png`, скопированная в `static`.

## Структура

- `app.py` — маршруты Flask и генерация DOCX;
- `site_data.py` — контент сайта;
- `templates/` — страницы и общие части;
- `static/css/style.css` — оформление;
- `static/js/main.js` — интерактив сценариев, FAQ и калькуляторов;
- `static/docs/` — документы пользователя;
- `static/img/journeys/` — блок-схемы.

## Запуск

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска открыть:

```text
http://127.0.0.1:5000/
```
